## Пульсар (Frontend)
Frontend для веб-приложения Пульсар на React JS

## Функциональность
- Авторизация через JWT
- Управление задачами — создание, редактирование, удаление
- Админ-панель для управления пользователями и задачами
- Поиск и фильтрация задач
